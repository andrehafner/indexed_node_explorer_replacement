Wallet Configuration

To use the wallet features, you need to configure your Ergo node API key:

  1. Find your API key in your node's ergo.conf file under scorex.restApi.apiKeyHash
  2. Add the unhashed API key (not the hash) to your docker-compose.yml:
environment:
  - NODE_API_KEY=your_api_key_here

The API key is the plain text password, not the hash shown in your node config.
